# History Practice



Короткий опис: Це навчальний проєкт для практики роботи з Git.



## Функціональність

- Створення і коміти файлів

- Відміна змін

- Перегляд історії змін

- Робота з staged та untracked файлами



## Встановлення
1. Клонуйте репозиторій:
``````bash
git clone https://github.com/ваше-ім’я/history-practice.git
``````
2. Перейдіть у папку проєкту:
``````bash
cd history-practice
``````
3. Запустіть програму:
``````bash
python calculator.py
``````
